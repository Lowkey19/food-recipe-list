# food-recipe-list
Display food recipe list

How to run:

Client:
- Go to food-recipe-app/client
- yarn -> install dependecies
- yarn start -> run client

Server:
- Go to food-recipe-app
- Create .env from .env.example
- yarn -> install dependecies
- yarn start -> run server
